# Lokalizace JARVIS Windows

Tento adresář je jediným zdrojem pravdy pro průběh české lokalizace.

## Dokumenty

- [DECISIONS.md](../../DECISIONS.md) — rozhodnutí s dopadem na celý projekt.
- [PLAN.md](PLAN.md) — jednotlivé fáze, kroky a jejich aktuální stav.
- [CHECKLIST.md](CHECKLIST.md) — povinné kontroly každé změny.
- [HISTORY.md](HISTORY.md) — chronologická historie provedených kroků.
- [GLOSSARY.md](GLOSSARY.md) — sjednocené české termíny a nepřekládané hodnoty.

## Pracovní postup

1. Zkontroluj platná rozhodnutí v `DECISIONS.md`.
2. Vyber nejbližší krok se stavem `TODO`.
3. V `PLAN.md` jej změň na `IN PROGRESS`.
4. Proveď úpravu a příslušné kontroly.
5. Aktualizuj kontrolní body daného kroku.
6. Přidej nový záznam na konec `HISTORY.md`.
7. Krok označ jako `DONE` pouze tehdy, když je implementace ověřená.

Při objevení nové práce ji nejprve přidej do plánu. Nezdokumentované změny se
nepovažují za dokončené.
