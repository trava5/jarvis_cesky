# 001_weather

Ověřená akce pro aktuální počasí a krátkou předpověď.

## Soubory

- `weather.py` — implementace funkce `get_weather_summary`.

## Nástroj

Agent používá tuto akci přes nástroj `get_weather`, který je popsaný v
`actions/tool_catalog.py`.
